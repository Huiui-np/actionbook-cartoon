# 《仙逆》年番舆情汇总 —— 需求文档

## 1. 目标

在各大社交平台搜索「仙逆年番」相关视频/话题，爬取评论区高热度评论，截图留存，最终汇总成 Excel 表格输出。

## 2. 平台范围

| 优先级 | 平台 | 搜索关键词 |
|--------|------|------------|
| P0 | 抖音 | 仙逆 年番 |
| P0 | B 站（哔哩哔哩） | 仙逆 年番 |
| P0 | 微博 | 仙逆年番 |
| P1 | 小红书 | 仙逆 年番 |

## 3. 输出格式

Excel 文件，表名为「《仙逆》年番舆情汇总」，结构如下：

| 列 | 说明 |
|----|------|
| 日期 | 舆情发现的日期 |
| 渠道 | 来源平台（抖音 / B 站 / 微博 / 小红书） |
| 内容 | 舆情分类及评论摘要：<br>• 正向/中性 (X%)：列出用户好评内容<br>• 负向 (X%)：列出用户差评内容 |
| 截图/链接 | 评论截图拼接或原帖链接 |

### 3.1 内容列规范

每条舆情记录按**情感倾向**分为两类：

- **正向/中性**：用户好评、期待、赞赏等
- **负向**：用户吐槽、不满、批评等

每类下面用编号列出具体评论内容，格式示例：

```
1. 网友表示，本集剧情精彩王林杀伐果断不圣母很喜欢
2. 网友表示动漫剧情合理节奏很棒
3. 网友表示期待王林二次化凡的剧情
```

### 3.2 截图列规范

- 每条评论对应一张截图（PNG/JPG）
- 同一舆情记录的多张截图横向拼接后放入该单元格
- 截图需清晰展示：用户名、头像、评论内容、点赞/回复数

## 4. 执行流程

1. **搜索**：在各平台搜索「仙逆年番」相关视频/话题/帖子
2. **筛选**：选取播放量/热度较高的内容（如 Top 5-10）
3. **爬取评论**：对每条内容爬取评论区前 10-20 条评论
4. **分类整理**：将评论按正向/中性/负向归类，提取核心观点
5. **截图**：对代表性评论截图保存
6. **汇总**：写入 Excel 表格，生成最终报告

## 5. 技术实现方式

使用 Actionbook 浏览器自动化引擎：

- `actionbook browser start` — 启动浏览器会话
- `actionbook browser goto` — 导航到目标平台
- `actionbook browser snapshot` — 获取页面结构
- `actionbook browser screenshot` — 截取评论截图
- 使用 Python（openpyxl）将数据写入 Excel

## 6. 注意事项

- 各平台评论区结构不同，需分别适配
- 截图需包含完整评论内容，避免截断
- 情感分类需客观，不带有主观判断
- 同一评论可能被多个平台用户讨论，需去重或标注来源
